# Gotchu

> **Status: superseded prototype**

Gotchu was an early exploration of a persistent AI companion and virtual-creature product inspired by digital pets, collectible characters, and relationship-based interaction loops.

This repository is **not an active production project**. Its recent commit history reflects an unsuccessful source-archive recovery attempt rather than the current application code. Active product development continued separately under the private Nymbi project.

## Original product direction

The prototype explored:

- a persistent creature with memory and evolving personality
- care, play, and conversation loops
- collectible visual traits and progression
- lightweight daily rituals designed for emotional continuity
- an AI layer bounded by a consistent character identity

## Why this repository remains public

It is retained as an honest record of an early product experiment and a failed recovery workflow. It should not be used to evaluate the current build quality, architecture, or reliability of the active product.
